# Trendy Fits

 * This application helps the process of product checkout at each level in a e-commerce store. 

 * This application scans the QR of the product and helps updating the status at each of the following levels.
    * Pending
    * Processing
    * Cad
    * Cutting
    * Sewing
    * Finishing
    * Ready to Ship
    * Delivered
    * Shipped
